# AI健康饮食助手

## 项目概述
AI健康饮食助手是一个专注于个性化、场景化、轻量化健康饮食指导的应用程序。系统通过AI算法结合用户身体数据、饮食偏好、健康目标，提供定制食谱、食材推荐、饮食打卡、营养科普等一站式服务。

## 功能特性
- 个人健康档案管理
- 个性化食谱推荐
- 饮食记录与营养分析
- 健康目标跟踪
- 营养科普与建议

## 技术栈
- 后端: Spring Boot 3.x, Java 21
- 数据库: MySQL
- 前端: Vue 3, JavaScript
- 构建工具: Maven, Vite

## 环境要求
- Java 21+
- Maven 3.6+
- MySQL 8.0+
- Node.js 16+
